calculator

This exercise had me make a simple calculator using prompt(), and variables.
The requirements were to make a program that does the following:

  1. Prompts the user for a first number.
  2. Stores that first number in a variable.
  1. Prompts the user for the second number.
  2. Stores that number and responds with the SUM by using an alert. 
                      
                      var firstNum = prompt("Enter a number");
                      var secondNum = prompt("Enter another number");
                      var numOne = Number(firstNum);
                      var numTwo = Number(secondNum);

                      alert("The sum is: " + (numOne+numTwo));

BONUS: Make a program that can subtract, multiply, and also divide! 

                      
                      var firstNum = prompt("Enter a number");
                      var secondNum = prompt("Enter another number");
                      var choice = prompt("What would you like to do? \nADD? \nSUBTRACT? \nMULTIPLY? \nDIVIDE?");
                      var answer = choice.toLowerCase();


                      if(answer==="add") {
                          var results = Number(firstNum) + Number(secondNum);
                          alert(results);
                      }
                      else if(answer==="subtract") {
                          var results = Number(firstNum) - Number(secondNum);
                          alert(results);
                      }
                      else if(answer==="multiply") {
                          var results = Number(firstNum) * Number(secondNum);
                          alert(results);
                      }
                      else if(answer==="divide") {
                          var results = Number(firstNum) / Number(secondNum);
                          alert(results);
                      }

keyless car

Make a keyless car! This car will only let you drive if you are over the age of 18.
For this exercise I had to make it do the following:

  • Using prompt() and alert(), ask a user for their age.
  • If they say they are below 18, respond with:
    "Sorry, you are too young to drive this car. Powering off"
  • If they say they are 18, respond with:
    "Congratulations on your first year of driving. Enjoy the ride!"
  • If they say they are over 18, respond with:
    "Powering On. Enjoy the ride!" 
                      
                      var age = prompt("How old are you?");
                      var ageResult = Number(age);

                      if(ageResult < 18) {
                          alert("Sorry, you are too young to drive this car. Powering off");
                      }

                      else if (ageResult === 18) {
                          alert("Congratulations on your first year of driving. Enjoy the ride!");
                      }

                      else if(ageResult > 18) {
                        alert("Powering On. Enjoy the ride!");
                      }

Although this works, unfortunately it would run immediately upon page load and any time the page refreshes. Also, it isn't really what I would call reusable. This will get somewhat solved in the following exercises.
